# 디자인 피드백 요청 양식 (PO 공유용)

디어가 슬랙 채널이나 개인 DM 안내에 붙여 쓰는 템플릿. PO가 이 양식으로 요청하면 레벨 판정에 필요한 3요소가 바로 확보된다.

---

## 슬랙 공유용 스니펫

```
프로토타입 디자인 피드백이 필요하시면 아래 4줄과 함께 보내주세요!

1. 링크: (Figma나 프로토타입 링크)
2. 누가 쓰나요?: (일반 유저 / 셀러·파트너 / 브랜드사 / 사내 어드민)
3. 얼마나 쓰나요?: (예: 일 1~2회 운영용, 홈 진입 전체 트래픽 등 대략이면 돼요)
4. 잘못되면 뭐가 깨지나요?: (예: 정산 금액 틀어짐, 단순 조회라 리스크 낮음 등)

이 정보로 어디까지 볼지(안전선만 / 흐름까지 / 풀 리뷰) 정하고, 고쳐야 하는 것과 참고만 하면 되는 걸 나눠서 회신드려요.
```

## 회신 시 태그 안내 (처음 요청한 PO에게 한 번만)

```
피드백은 세 가지 태그로 나눠 드려요.
- [이것만 고치면 돼요]: 반영 안 하면 내보내기 어려운 것
- [제안]: 반영 권장, 판단은 PO 몫
- [참고]: 참고만 하셔도 돼요
```

## 운영 메모

- 양식 없이 온 요청도 거절하지 않는다. 이번 건은 그대로 처리하고, 회신 끝에 "다음엔 이 4줄과 같이 주시면 더 빨라요"로 양식 안내
- 4줄 중 빠진 정보는 디어/Claude가 추론으로 채우되 회신에 전제를 명시 ("어드민 저사용 기준으로 봤어요. 아니면 알려주세요")
