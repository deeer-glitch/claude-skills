# Output Modes — 4가지 출력 형식

발화에서 자동 추론. 모호하면 한 줄 확인.

## 1. 풀 페이지 (Full Page)

KV + 섹션 N개 + CTA로 *한 화면 흐름*을 보여줌. 온보딩·랜딩 페이지 구조 합의용.

**언제 사용**
- "온보딩 라이트화 풀 페이지 와이어"
- "{화면} 구조 와이어"
- "{기능} 페이지 전체 와이어"

**구성 요소 (위→아래)**
1. GNB (자리만)
2. KV — `.kv` 다크 영역. 메인 카피 + 사회적 증거 카드 + CTA
3. 섹션 N개 — `.sec`/`.sec.alt` 번갈아. 각 섹션 = 헤더 + 본문 (조건 보드/브랜드 row/스텝 등)
4. 하단 CTA — `.bot-cta-wrap` + `.bot-cta`
5. 주석 박스 (선택) — `.anno` 1~3개 (`A1` 새 구조 / `A2` 변경 / `A3` 미확정)
6. (옵션) Empty 가이드 별도 패널 — `.empty-section`
7. Footnote

**샘플**: `samples/light-structure-fullpage.html`

## 2. N-State 비교 (State Compare)

같은 화면의 *다른 상태/옵션* 2~4개를 나란히. State 1·2·3 각각 모바일 device로.

**언제 사용**
- "State 비교 시안"
- "옵션 A·B·C 보여줘"
- "v3 만들어줘" (옵션 비교가 흔한 v3 패턴)

**구성**
- 메타 헤더 + 변경 노트
- 섹션 헤더 (예: "A. 브랜드 커뮤니케이션 3안 비교")
- `.cols` 안에 N개 device 나란히 — 각 device 위에 옵션 라벨(`.col-label .opt-tag`)
- 옵션 비교 매트릭스 (`.compare` 테이블)
- 별도 섹션 — 카드 디테일·인터랙션 (스낵바 등)
- 주석 박스 (`A1` 데이터 정합 / `A2` 변경 / `A3` 결정 항목 / `A4` 정책 / `A5` 엣지)
- Footnote

**샘플**: `samples/v3-wireframe-state-compare.html`

## 3. 컴포넌트 비교 (Component Compare)

*같은 컴포넌트의 다른 형태* — 모듈 헤더·카드 같은 부분 단위 옵션. State 비교보다 작은 단위.

**언제 사용**
- "로고 노출 옵션 비교"
- "헤더 형태 3안"
- "{컴포넌트} 옵션 A/B/C"

**구성**
- 짧은 메타 + 한 줄 설명
- `.cols` 안에 옵션 패널 N개 — 각 패널 = `.logo-opt` 또는 비슷한 박스에 *해당 컴포넌트만* 보여줌 + 한 줄 설명
- (옵션) 비교 매트릭스
- 주석 박스 — 데이터 정합 / 추천 한 줄

**언제 풀 페이지가 아닌 이걸 쓰나**
- 화면 전체가 아닌 *한 부분*의 옵션 비교
- 모바일 device 프레임도 필요 없을 수 있음 (헤더 영역만 띄움)

## 4. 메타포 설명 (Metaphor Explain)

추상 개념을 *그림·이모지·박스*로 풀어 설명. 정책 차이·데이터 흐름·UX 결정 근거 같은 *비시각적 개념*을 비기술자/주니어에게 설명할 때.

**언제 사용**
- "이거 그림으로 설명해줘"
- "메타포로"
- "{개념} 왜 그런지 시각화"

**구성**
- 상황 박스 — 이모지 + 짧은 설명 (예: 🛒 장바구니 / 📦 패키지 — 침대 A·B로 셋업 보여주기)
- ❌ Bad case vs ✅ Good case 두 케이스 박스
- 각 케이스 안 — 단계별 동작 + verdict (`😢 못 봄` / `😊 보임`)
- 한 줄 요약 박스

**디자인 톤**
- 모바일 device 안 띄움 (개념 설명이라)
- 이모지 활용 OK (🛏️ 🛋️ 📦 🛒 등 — 사물 표현)
- 색 — `.verdict.bad` 빨강 옅음 / `.verdict.good` 초록 옅음

## 모드 추론 우선순위 (모호한 발화 시)

| 발화 키워드 | 추론 모드 |
|---|---|
| "풀 페이지", "전체", "랜딩" | 풀 페이지 |
| "비교", "옵션", "A/B/C", "v3" | State 비교 또는 컴포넌트 비교 |
| "그림으로", "메타포", "왜 그런지" | 메타포 설명 |
| "옵션 비교" 중 *컴포넌트 하나만* | 컴포넌트 비교 |
| "옵션 비교" 중 *화면 전체 상태* | State 비교 |
| 모호 | 한 줄 확인 ("State 비교로 갈까요, 컴포넌트만 비교할까요?") |
